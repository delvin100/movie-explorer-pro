# 🎬 Movie Explorer

A full-stack movie exploration platform that enables users to discover movies, manage watchlists, submit reviews, request movies, and interact through a secure authentication system. The application also includes an admin dashboard for managing users, reviews, and movie requests.

---

## 📖 Overview

Movie Explorer is a full-stack web application built using **Node.js**, **Express.js**, **MongoDB**, and **JavaScript**. It integrates with the **TMDB API** to provide real-time movie information while offering additional features such as user authentication, personalized watchlists, reviews, and administrative controls.

---

## ✨ Features

- 🔍 Search and discover movies
- 🔥 Browse trending and popular movies
- 👤 User registration and login with JWT authentication
- ❤️ Create and manage personal watchlists
- ⭐ Submit and view movie reviews
- 📥 Request new movies
- 🛡️ Admin dashboard for managing users, reviews, and requests
- 📱 Responsive user interface

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/delvin100/movie_explorer_updated.git
```

### Install Dependencies

```bash
npm install
```

### Start the Server

```bash
npm start
```

Open your browser and visit:

```
http://localhost:3000
```
